Подключение

Зайдите в настройки Вашей группы, в раздел Работа с API

Зайдите в раздел Ключи доступа. Нажмите кнопку Создать ключ. Выставьте все галочки и нажмите Создать. Полученный ключ сохраните до дальнейшего использования.

Зайдите в раздел Callback API
- В подразделе Настройки сервера Версию API укажите 5.103
- Адрес укажите https://botnet-daemon.herokuapp.com/api/callback/{{ bot.hash }}
- Снизу в рамке копируете те цифры, что идут после group_id. Это ID Группы, сохраните его себе.
- Строку, которую должен вернуть сервер также сохраните себе.
- Секретный ключ задайте по своему усмотрению и также сохраните.
- В подразделе Типы событий поставьте галочку на Входящее сообщение.

Нажмите Редактировать и укажите Ключ доступа, код подтверждения, секретный ключ и ID группы те, что Вы получили ранее.
